Theme ***DATA MANAGEMENT AND SHARING*** "@OMD". TOPIC 5 ***SPREAD SHEET I*** "24 PERIODS". Competency; The learner uses spreadsheet software (e.g. MS Excel) to create, save, edit and print spreadsheets. Learning Outcomes; The learner should be able to; a) use spreadsheet functions to manipulate data and print it, (k, s, v); b) use charts to represent data (s, v). • Open a spreadsheet. • enter data in a spreadsheet. • study the spreadsheet interface and try out the various features on text. • study a pre-type set hardcopy document with various formats and comment on the observed features. • use a simple spreadsheet formula to compute sum, min, max, average and count. • sort data using alphabetical, ascending & descending order. • generate simple labelled charts: pie, line and bar. • use a discovery approach to creatively come up with different formats of presenting information;
